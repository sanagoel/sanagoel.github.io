Describing the Data

Name: This is a Categorical text variable as the name of the video game is a label which can be used to catagorize but not for any arithmatics.
Platform: This is a Categorical nominal variable as platforms are lables but do not have any inherent order but games can be grouped based on platform.
year : This is a discrete Numerical variable as the year is a number and takes integer values rather than any fractional Value.
Genre: This is a Categorical nominal variable as Genre is a label and has no numeric value or order but you can categorize games by Genre.
Publisher: This is a Categorical nominal variable as the publisher is a label with no numberic value but games can be catagorized based on this, similar to genre and platform.
NA_sales, EU_sales, JP_sales, Other_sales and Global_sales: This is a continuous Numerical variable as sales can take any vlaue within this range and arithmatic operations are valueable in this feild like SUM or Mean.
